Authentik = the passport office

Owns identity: creates users, manages groups, enforces MFA, and issues signed OIDC tokens. Knows who everyone is. Does not decide what they can access — that's Pomerium's job.

Pomerium = the bouncer at every door

Trusts Authentik's wristband, then applies its own policy.yaml rules on every single request. Can deny access based on time, device, route — regardless of what Authentik said about identity. Policy lives in your repo as code.

Two-model critic pattern

Co-pilot proposes. Auditor independently reviews for architectural integrity, ES|QL query cost, and security risk — before any output reaches a human or the CI pipeline.

Agent behavior as a detection source

Every MCP tool call is logged to Elasticsearch. Detection rules fire on anomalous agent behavior — prompt injection, out-of-scope tool calls, auditor pass rate spikes. The platform watches its own AI.
